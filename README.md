## How to Use
1. Download the AGIQA-3k dataset and place the images and MOS file in your project directory.  
2. Open `TiQE.ipynb` in Jupyter Notebook or Google Colab.  
3. Update the dataset path inside the notebook if needed.  
4. Run the notebook to reproduce training, testing, or inference.
